## 🚀 Quick Start (5 Minutes)

### Step 1: Download
```bash
git clone https://github.com/ravikumarve/study-assistant.git
cd study-assistant
```

### Step 2: Install
```bash
./setup.sh
```

### Step 3: Run
```bash
python app.py
```

### Step 4: Learn
Open your browser → `http://localhost:5000`

**That's it. No account. No credit card. No BS.**
